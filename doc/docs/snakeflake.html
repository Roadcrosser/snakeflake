class snakeflake.snakeflake.Snakeflake(timestamp, serial, machine_id, epoch=None, constants=None, snakeflake_id=None)[source]¶

Defines a snakeflake

This class holds the attributes of a snakeflake and calculates the actual snakeflake ID.

Note

When instancing this class, please try to use the from_generator() or from_snowflake() classmethods to generate snakeflakes instead.

Parameters

  • timestamp (datetime.datetime) – The timestamp of the snakeflake.

  • serial (int) – The serial number of the snakeflake.

  • machine_id (int) – The machine ID of the snakeflake.

  • epoch (datetime.datetime) – When this snakeflake started counting from.

  • constants (SnakeflakeConstants) – The constants this snakeflake uses.

  • snakeflake_id (int) – The ID of this snakeflake. This should only be specified if you are deriving information from a snakeflake ID.

calculate()[source]¶

A generic “smart” calculation method that runs either calculate_snakeflake() or reverse_calculate_snakeflake() depending on the current attributes set.

calculate_snakeflake()[source]¶

Calculates a snakeflake.

This method calculates and updates the snakeflake_id attribute based on the timestamp, serial, and machine_id attributes set upon initialization.

classmethod from_generator(timestamp, generator)[source]¶

Generates a snakeflake using data from the generator object. The snakeflake ID itself will not be calculated until calculate_snakeflake() is run.

Warning

This will not increment the serial number of the generator itself. Please instead generate snakeflakes from the SnakeflakeGenerator directly for this functionality.

Parameters

  • timestamp (datetime.datetime) – The timestamp of the snakeflake.

  • generator (SnakeflakeGenerator) – The generator with the data that will be used to generate the snakeflake.

classmethod from_snakeflake(snakeflake_id, epoch=None, constants=None)[source]¶

Generates a snakeflake from a snakeflake ID. This is usually used for deriving the timestamp of a snakeflake, although the serial and machine IDs will also be calculated.

Note that the resulting attributes will not be calculated until reverse_calculate_snakeflake() is run.

Parameters

  • snakeflake_id (int) – The snakeflake.

  • epoch (datetime.datetime) – When this snakeflake started counting from.

  • constants (SnakeflakeConstants) – The constants this snakeflake uses.

class snakeflake.config.SnakeflakeConstants(timestamp_bits, timescale, serial_bits, machine_id_bits)[source]¶

The Snakeflake Config class that deals with constants

It is recommended to initialize this class using the defaults() classmethod instead.

Parameters

  • timestamp_bits (int) – The number of bits to allocate for the timestamp.

  • timescale (int) – The timestamp precision in microseconds.

  • serial_bits (int) – The number of bits to allocate for the serial number.

  • machine_id_bits (int) – The number of bits to allocate for the machine ID.

classmethod defaults()[source]¶

Initializes the constants with their default values.

These default values were taken from https://github.com/sony/sonyflake.

class snakeflake.config.SnakeflakeGeneratorConfig(epoch=None, machine_id=None, constants=None, timestamp_method=None)[source]¶

The class that holds the configuration Snakeflake Generator Config class
